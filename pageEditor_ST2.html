Challenges and Lessons Learned

1. Ensuring Responsive Design

Maintaining a consistent layout across different screen sizes was challenging. Using flexbox and media queries helped create a responsive design, but some sections required additional adjustments to prevent overlapping elements.

2. Navigation Menu Behavior

The hamburger menu implementation required JavaScript to toggle visibility, and ensuring smooth transitions between mobile and desktop views was tricky. Debugging event listeners and styling hidden elements improved functionality.

3. Accessibility Considerations

Ensuring compliance with WCAG 2.1 guidelines required reviewing alt attributes, color contrast, and keyboard navigation. Some color choices had to be adjusted for better readability.

4. Page Load Optimization

Using multiple high-quality images and animations affected page load speed. To improve performance, image compression and lazy loading techniques were considered.

5. Custom Animations and Interactivity

The floating bubble animation added visual appeal, but required fine-tuning with CSS @keyframes and JavaScript for smooth rendering. Performance optimizations were needed to prevent lag on lower-end devices.

6. Internal Page Linking Issues

Implementing anchor links for smooth scrolling had some challenges due to incorrect id references. Ensuring accurate linking and adding scroll-behavior: smooth; in CSS resolved the issue.
