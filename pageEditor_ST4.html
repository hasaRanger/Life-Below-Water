User Profile Page

Technical Description

The User Profile page was built using HTML and CSS grid/flexbox for layout structuring. The User profile contains sections like user details, description, skills, goals, frustrations, UN goals, experience, and contact.

HTML

  • Buttons

    This is used to call JavaScript functions <button class = "scroll_button" onclick="scrollUp()">↑</button>

  • iframes

    Google Calendar is embedded using an iframe <iframe id = "calendar" src="...." style="border:solid 1px #777" width="800" height="300" frameborder="0" scrolling="no"> </iframe>

CSS

  • CSS Grids and flexbox

    CSS grid/flexbox for layout structuring display : grid; and display : flex;

  • Hover

    Hover effects to improve user interactivity .goals:hover { declaration block }

JavaScript

  • Scroller

    To add scroller functionality function scrollUp() { document.getElementById("scroller").scrollBy({top : -400,behaviour : "smooth"}); }

  • alert

    JavaScript to trigger alert box alert("Your inquiry has been submitted successfully!");

Accessibility

To improve accessibility on the User Profile page, the following measures were implemented:

  • Semantic HTML elements (e.g., <section>, <header>, <footer>) for screen reader compatibility.
  • Alt text for all images, ensuring visually impaired users can understand the content.
  • Keyboard navigability, allowing users to interact with all elements without a mouse.
  • High-contrast color schemes for readability and better visual clarity.

Key Techniques Used

  • Grid layout for responsiveness
  • Hover effects & transitions for better user interaction
  • JavaScript functions for form handling and scrolling

Site Map Page

Technical Description

The Site Map page uses SVG graphics to visually represent the website structure. It employs CSS styling and transitions for a smoother user experience.

HTML

  • SVG

    This element is used to create scalable vector graphics in HTML <svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="yellow" /> </svg>

  • hyperlinks

    To provide a link to another website <a href = "https://www.google.com/"></a >

CSS

  • Hover

    Colour change on hovering .circle:hover { fill: #94D2BD; }

  • CSS Grids

    CSS Grid to create a container that seamlessly accommodates the SVG. display : grid;

Accessibility

To ensure accessibility on the Site Map page, the following enhancements were made:

  • Keyboard accessibility ensures that users can navigate the site map using the Tab key.
  • Hover effects are accompanied by focus states for better usability across different input methods.

Key Techniques Used

  • SVG elements for graphical representation
  • Flexbox & Grid for layout alignment

Content Page

Technical Description

The Content Page features news updates presented using a grid-based layout. It includes dropdown navigation, images, and external news links.

HTML

  • Internal Linking

    Implementing internal linking for seamless navigation within the same page. <a href="#Ocean_conference">Mobilizing for the 2025 UN Ocean Conference</a> Destination location is marked with id = "Ocean_conference"

  • Dropdown Menu

    Enabling navigation to any location within the website. The menu can be displayed by hovering over the button for an interactive experience.<button class="dropbtn">☰</button> >

CSS

  • Cards

    Seperate card for each news. individual cards ensure the easy updatability for future updates.

  • CSS Grids and Flexbox

    CSS Grid and flex box used to structure news cards. display : grid; and display : flex;

Accessibility

The Content Page includes the following accessibility features:

  • All images contain alt text.
  • Dropdown menus are accessible using keyboard navigation.
  • Font sizes and spacing ensure readability for all users.

Challenges and Lessons Learned

Challenges

  • Aligning SVG elements in the Site Map page.
    • The alignment of SVG elements on the Site Map page was optimized by adjusting coordinates and the viewBox to ensure proper positioning and scalability.
  • A smooth scroller integration into the Skill Card section of the User Profile page.
    • A smooth scrolling feature was integrated into the Skill Card section of the User Profile page using JavaScript, allowing seamless navigation through the listed skills.

Lessons Learned

  • CSS positioning plays a crucial role in SVG usability.
  • JavaScript event listeners enhance user interaction.

Compliance

Ensuring compliance with web standards and accessibility guidelines is essential for a user-friendly and legally sound website. The following principles were adhered to:

  • HTML & CSS Standards: All pages follow HTML5 and CSS3 best practices, ensuring proper structure and styling.
  • Accessibility Compliance: Implemented semantic HTML, ARIA attributes, and high contrast color schemes for better readability.
  • Responsive Design: Used flexible layouts and media queries to provide a seamless experience across different screen sizes.
  • Privacy & Security: Ensured form submissions do not store sensitive user data, complying with basic GDPR principles.
  • SEO Best Practices: Structured data, and descriptive alt text were used to improve search engine visibility.

By following these principles, the website ensures compliance with modern web regulations and enhances user experience.
