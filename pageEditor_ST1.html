Splash Page

Introduction

Our website is designed to spread awareness and conservation efforts related to Life Below Water, aligned with the United Nation's Sustainable Life Development Goal 14.As student 1 I was tasked with designing and implementing multiple pages for this website. The pages I created include:

  • Splash Screen-The first page that users are first greeted with that will autmatically redirect to the home page after a few seconds
  • Volunteer Page-A page inlcuding information about volunteer programs and user reviews
  • Content Page-A page providing fun facts about the ocean,marine life and ecosystem

Technical Description

HTML structure

Used HTML elements such as <section>, <div> and <h1>to properly organize and section the content. The meta refresh tag is added to automatically redirect the user to the home page after 4 seconds. Also a loading animation is implemented to make the page more visually engaging.

CSS Styling

To enhance the theme, applied an oceanic background video using the<video> tag. Used keyframe animations(@keyframe)to create a floating bubble effect for a more immersive experience. Designed a water-text effect with -webkit-background-clip: text; to make the “loading” text blend in with the background and the theme. Additionally, implemented a fade out transition (.fade-out) to make the content seemingly fade out before redirecting to the home page to make the experience visually smoother.

JavaScript

The delayed fade out effect (setTimeout) to create a fade-out transition before redirecting Ensured elements disappear smoothly by modifying their opacity and display.

Accessibility

For text readability and contrast the title and mission statement used bright colors against a dark background for better visibility and used large font sizes to make sure even in small screens the text is readable. All images have descriptive alt attributes and the “Loading” text is inside a <div> ensuring screen reader can detect it.
prefers-reduced-motion is implemented in CSS to make sure users who has disabled animation in system settings will experience a static version of the splash screen. And ease-in-out animations is used to prevent sudden movements.

Volunteer Page

Technical Description

HTML Structure

Volunteer programs are displayed in a structured card layout with an image description and buttons for more information. <select> dropdown menu allows users to sort programs according to filters such as age, most recent and more. A star rating implemented using <div class="stars"> lets users rate programs interactively. Furthermore, a table displays a breakdown of user ratings under the topics of impact, support, enjoyment and safety. Users can also write reviews and submit feedback through text area and submit button.

CSS Styling and Design Choices

For the background used an ocean themed gradient color background to suit the theme of life below water. High contrast text is used for readability on both light and dark backgrounds. Also to make important buttons such as “Apply Now” stand out colored them with contrasting colors such as green and aqua.

Animation and hover effects

Volunteer cards expands on hover to display more details by using .cards:hover and increasing the height enough to display the content. smooth transition applied to buttons and links for better user interaction. Volunteer cards and review cards also have a subtle glow effect on hover by using box-shadow tag.

JavaScript Functionality

All star elements are selected using document.querySelectorAll(".stars div") then added a click event listener to each star to detect selection. To ensure that clicking a star activates it and all previous ones a loop is used to check index positions (index1>=index2) then .active class is removed from stars that comes after the selected ones. For cards when the mouse is no longer hovered over them, to automatically scroll up to the top an event listener is added to detect when the mouse is no longer being hovered over card.scrollTo({ top: 0, behavior: "smooth" }); is used to scroll upback to the top.

Accessibility

  • High contrast colors and large fonts are used for texts for better readability.
  • Volunteer cards have smooth hover transition effects using ease-in-outavoiding a sudden transition.
  • To disable hover animations for users who are sensitive to motion the prefers-reduced-motion tag is used.

Content Page

Technical Description

HTML Structure

For altering layouts contents are organized into sections such as <section class="contentleft">, <section class="contentright">. Navigation links (eg:- a href=”#ocean”) allow users to navigate through different sections of the page easily. Using images and text blocks effectively ensures a balanced and structured presentation of information.

CSS Styling

Color theme

To match the ocean theme used a dark blue for the background and for better readability of texts added contrast-enhancing text colors.

Animation

Implemented fade-in animations for titles and slide-in animations for images and info blocks using @keyframes for a unique scrolling experience. To make sure animations only trigger when elements enter the viewport applied animation-timeline:view();

Typography and layout

Used a combination of elegant and playful fonts such as "Sail", "Delius" and "Shadows Into Light" for a modern look. Structured content into left-right alternating sections for a smooth reading flow.

JavaScript

Using window.scrollTo({ top: 0, behavior: "smooth" }); a button is created to scroll to the top of the page when the button is clicked and the behaviour:"smooth" makes sure the scrolling to the top is smoother rather than instantly going to the top of the page.

Accessibility

  • Light colors are used for text to stand out from the dark colored background.
  • Alternative texts are available for every image
  • Disabled animations for users with motion sensitivity by implementing prefers-reduced-motion
  • Entry animations for appear_from_left,appear_from_right is smooth and not sudden or disruptive.

Challenges and Lessons Learned

Challenges Solution
The title of the page would be hard to read when certain ares of the background video comes to the background In CSS implemented mix-blend-mode: difference to chnage the color according to the background
Loading text animation losing the lower fill at the end of the animation Adjusted background-size and -webkit-radial-gradient properties to maintain fill consistently
Redirect felt too sudden and abrupt Implemented a fade out effect right before the page redirects for better user experience.
Sorting dropdown was not properly aligned Used flexbox to align it next to the heading
Volunteer card hover effect was too sudden Adjusted transition:0.5s ease-in-out; for smoother scaling.
For the table to look stylish it needed to change the colors of the rows alternatively but had to change the properties of each row one by one. Used tr:nth-child() and put according values(odd or even) to easily change the colors of the rows.
In the cards when the mouse is not hovered over them and the height returns to normal it would still be stuck at the bottom part. Using JavaScript added an event listener to detect when the mouse is no longer hovered over the cards for the content to automatically scroll back up to the top.
Scroll animations weren’t triggering properly. Used animation-timeline= view() to activate animations when only elements enter the viewport.
